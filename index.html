Google Apps Script 이메일 알림 가이드
이메일 알림은 보안상 백엔드 서버가 필요합니다. Google Apps Script를 사용하면 무료로 자동 이메일 발송을 구현할 수 있습니다.
1
알림 일정 데이터 내보내기
알림 설정 후 "CSV 내보내기" 버튼을 클릭하여 알림 일정 파일을 다운로드합니다.
이 파일을 Google Sheets에 업로드합니다.
2
Google Apps Script 연동
Google Sheets → 확장 프로그램 → Apps Script 열기
아래 스크립트를 붙여넣고 이메일 주소를 수정합니다.
Apps Script 예시 코드
function checkAndSendAlerts() {
  const sheet = SpreadsheetApp.getActiveSheet();
  const data = sheet.getDataRange().getValues();
  const today = new Date();
  const email = "your@email.com"; // 수신 이메일

  data.slice(1).forEach(row => {
    const [project, item, deadline, submitTo, daysLeft] = row;
    const dl = new Date(deadline);
    const diff = Math.ceil((dl - today) / 86400000);
    if ([30, 14, 7, 3, 1].includes(diff)) {
      GmailApp.sendEmail(email,
        `🔔 [SafeGuard] ${item} D-${diff} 알림`,
        `프로젝트: ${project}\n항목: ${item}\n` +
        `제출기한: ${deadline}\n제출처: ${submitTo}\n` +
        `남은 기간: ${diff}일`
      );
    }
  });
}

// 매일 오전 9시 트리거 설정:
// 편집 → 트리거 → 함수: checkAndSendAlerts → 시간 기반 → 매일
3
